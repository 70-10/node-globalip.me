# globalip.me

## Install & Usage

```
$ npm install globalip.me
```

```js
const globalip = require("globalip.me");

globalip().then(ip => console.log(ip));
// => 100.100.0.1
```

### CLI

```
$ npm install -g globalip.me
$ global-ip
100.10.0.1
```

#### Use npx

```
$ npx globaliip.me
100.10.0.1
```
